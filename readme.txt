A beautiful block that displays curated nonduality and spiritual quotes from traditions including Advaita Vedanta, Buddhism, Taoism, Christian Mysticism, Sufism, and more.

== Description ==

The Nonduality Spiritual Quotes Block brings timeless wisdom directly into your WordPress site. It features a curated collection of profound quotes from the world's great nondual and mystical traditions.

**Features:**

* Over 30 carefully selected quotes from renowned spiritual teachers
* Six tradition categories: Advaita Vedanta, Buddhism, Taoism, Christian Mysticism, Sufism, and Inspirational
* Filter quotes by category or display from all traditions
* "New Quote" button lets visitors discover fresh wisdom with each click
* Smooth fade transition between quotes
* Customizable appearance: font size, text alignment, and color scheme
* Shows author name and tradition category with each quote
* Fully responsive design
* Works beautifully in both the editor and on the front end

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/telex-nonduality-quotes` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the "Nonduality Spiritual Quotes Block" via the block editor.

== Frequently Asked Questions ==

= Can I filter quotes by tradition? =

Yes! Use the block sidebar to select a specific category such as Advaita Vedanta, Buddhism, Taoism, Christian Mysticism, Sufism, or Inspirational. You can also choose "All Categories" to display quotes from every tradition.

= Can visitors get a new quote? =

Yes, a "New Quote" button appears on the front end allowing visitors to cycle through quotes with a smooth fade animation.
